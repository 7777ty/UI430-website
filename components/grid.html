# Grid 网格系统

Grid简单用法

简介

  • 1.span属性可以用于设置每格的长度，长度的总和为24。
  • 2.gutter属性可用于设置间隔，单位为px。
  • 3.offset属性可用于设置空隙，其接受的值与span值一致。
  • 4.请确保每个g-row内的offset和span总和为24
