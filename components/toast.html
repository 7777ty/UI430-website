# Toast 吐司弹出

this.$toast简单用法

简介

  • 1.position属性用于设置弹出框的方向，仅接受'top','bottom','left','right'四种值。
  • 2.autoClose属性用于设定弹出框是否关闭，接受一个布尔值。
  • 3.autoCloseDelay属性用于设置弹出框的自动关闭事件，单位为秒。
  • 4.可通过closeButton属性的text值设置关闭按钮文本，此外它还可以接受一个回调函数，在点击关闭按钮后调用这个回调函数。
  • 5.enableHtml属性用于设定弹出框是否支持HTML，接受一个布尔值。

预览

上方弹出 中间弹出 下方弹出

代码实例


<style>
   .toastWrapper {
       z-index: 30 !important;
   }
</style>

<g-button @click="$toast('点击弹出提示',{position:'top',autoClose:false})">上方弹出</g-button>
<g-button @click="$toast('点击弹出提示', {position:'middle',autoCloseDelay:1})">中间弹出</g-button>
<g-button @click="$toast('点击弹出提示', {position:'bottom'})">下方弹出</g-button>

设置关闭按钮

预览

上方弹出

代码实例

<style>
    .toastWrapper {
        z-index: 30 !important;
    }
</style>

<g-button @click="onClickButton">上方弹出</g-button>

methods: {
            onClickButton() {
                this.$toast('你知道我在等你吗？', {
                    closeButton: {
                        text: '知道了',
                        callback: () => {
                            console.log('他说知道了')
                        }
                    }
                })
            }
        }

设置关闭按钮

预览

上方弹出

代码实例

<style>
    .toastWrapper {
        z-index: 30 !important;
    }
</style>

<g-button @click="onClickButton">上方弹出</g-button>

methods: {
            onClickButton() {
                this.$toast('<strong style="color:red">红色加粗字体 </strong>', {
                    enableHtml:true
                })
            }
        }
